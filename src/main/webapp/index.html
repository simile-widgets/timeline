Timeline

Timeline is a DHTML-based AJAXy widget for visualizing time-based events. It is like Google Maps for time information. Below is a live example that you can play with. Pan the timeline by dragging it horizontally.

Just like Google Maps, Timeline can be used with zero software installation, server-side or client-side.
